Introduction

Welcome to the Apartments API documentation page! This page provides information about the available endpoints for the API.

Endpoints

Endpoint HTTP Method Description
/ GET Retrieves a list of all apartments in the database.
/:id GET Retrieves information for a specific apartment with the given ID.
/ POST Creates a new apartment in the database.
/:id DELETE Removes an apartment from the database with the given ID.
/:id PUT Updates an existing apartment in the database with the given ID.
